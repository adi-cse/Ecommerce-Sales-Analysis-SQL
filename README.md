# 🛒 E-Commerce Sales Analysis (SQL Project)

## 📌 Overview
This project analyzes e-commerce sales data using SQL. It includes database design, data insertion, and advanced queries to extract business insights.

## 🧰 Tech Stack
- MySQL
- SQL

## 📊 Features
- Total sales calculation
- Top-selling products
- Customer spending analysis
- Monthly revenue tracking

## 🗂️ Database Schema
- Customers
- Products
- Orders
- Order Details

## 🚀 How to Run
1. Open MySQL Workbench
2. Run `schema.sql`
3. Run `data.sql`
4. Run `queries.sql`

## 💡 Key SQL Concepts Used
- JOIN
- GROUP BY
- Aggregation Functions (SUM, COUNT)
- ORDER BY

## 📈 Outcome
This project demonstrates how SQL can be used to analyze real-world business data and generate actionable insights.
